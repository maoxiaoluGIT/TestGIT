git reset
版本前进后退
.基于索引值[推荐] $ git reset --hard b72422c
.使用^[只能往后]  $ git reset --hard HEAD^  [^^^...] 
.使用~[只能往后]  $ git reset --hard HEAD~3
--soft 仅仅在本地库移动HEAD指针
--mixed 移动HEAD指针 重置缓存区
--hard 移动HEAD指针 重置缓存区 重置工作区

$ git reset --hard HEAD

删除找回
1 删除前，提交到本地库
删除后提交到本地库 git reset --hard [指针位置]
删除后未提交到本地库 git reset --hard HEAD
